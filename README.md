# CAP Framework

> 基于Java17的轻量化Web开发MVC框架，融合事件驱动属性系统与现代IoC容器能力。

本仓库主要包含以下核心模块：

- **cap-property** —— 事件驱动属性系统，支持属性发布、订阅与回调。
- **cap-bean** —— 轻量级IoC容器，支持自动注册、依赖注入及生命周期管理。
- **cap-context** —— 启动/运行期上下文，统一管理 Bean 与属性环境。
- **cap-boot** —— 一站式启动层，集成自动扫描、属性注入、自动配置等能力。
